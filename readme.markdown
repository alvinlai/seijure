# seijure

This is a quickstart project that gets you off running Compojure and JSON.

## Usage

### Install dependencies using Lein

lein deps

### Start the server

lein ring server

## Preconfigured routes:

GET "/"

GET "/hi/:name"

GET "/json-layouts"

POST "/post/:val"
